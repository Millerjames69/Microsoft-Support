Getting Help in Windows

You can search for help directly from the taskbar, use the built-in "Get Help" app, or visit the official Microsoft support website.

How to Get Help in Windows

Create Windows Installation Media

  1. Go to the official Windows download page.
  2. Download the Media Creation Tool.
  3. Run the tool and follow instructions to create a bootable USB drive or download an ISO file.
